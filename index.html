Swinburne University of Technology

Aims

This project-based unit focuses on developing your skills to build web applications using web technologies. You'll work individually and in teams to tackle real-world web development challenges, building proficiency in selecting and applying suitable tools and techniques. The unit also covers essential aspects of security, ethics, accessibility, and professional communication within web development.

Unit Learning Outcomes

  • ULO1: Identify web technologies' core components and functions.
  • ULO2: Implement a project plan using project management strategies.
  • ULO3: Apply team frameworks, communicate and work effectively within teams.
  • ULO4: Utilise database query techniques to manage data.
  • ULO5: Demonstrate the use of web development technologies.
  • ULO6: Explain fundamental security principles in web development.
  • ULO7: Assess ethical considerations and security practices.

Content

  • Core Front-End and Back-End Development
  • Project-Based Learning and Version Control
  • Project Planning and Project Management
  • Web Security, Ethics, and Accessibility
  • Database Query Techniques
  • Professional Communication and Presentation
  • Teamwork Framework
